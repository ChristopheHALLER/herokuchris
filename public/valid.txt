Node 


API REST
Imposé comme référence pour les échanges de données entre serveurs et clients.
style d'API




GET : chercher une ressource spécifié
POST : envoyer une entité vers la ressource indiqué
PUT : remplacer des elements dans une ressource
DELETE : supprime la ressource indiqué
